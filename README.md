## Development

### Install

```shell
yarn install
```

### Run

```shell
# Start keysd
keysd

# Start app
yarn server
yarn start
```

You can run multiple instances by specifying a different app name and port:

```shell
# Start keysd (Keys2)
keysd -app Keys2 -port 22406

# Start app (Keys2)
yarn server
KEYS_APP=Keys2 KEYS_PORT=22406 yarn start
```
